# Choreo (choreo)

WSO2 Choreo is an enterprise-grade Internal Developer Platform (IDP) and application orchestration platform that helps organizations build, deploy, manage, and observe APIs, microservices, integrations, and AI applications across multi-cloud Kubernetes infrastructure. Choreo combines API management, CI/CD, GitOps, observability, FinOps, and platform engineering into a single AI-guided experience.

## APIs

### Choreo API Management API

Programmatic management of the full lifecycle of APIs on the WSO2 Choreo platform. Covers organizations, projects, components, APIs, builds, and deployments.

- **Human URL:** [https://wso2.com/choreo/docs/](https://wso2.com/choreo/docs/)
- **OpenAPI:** [openapi/choreo-api-management-openapi.yml](openapi/choreo-api-management-openapi.yml)

### Choreo Developer Portal API

Enables API consumers to discover, evaluate, subscribe to, and consume APIs hosted on the Choreo platform. Supports applications, subscriptions, and OAuth 2.0 / API key credentials.

- **Human URL:** [https://devportal.choreo.dev/](https://devportal.choreo.dev/)
- **OpenAPI:** [openapi/choreo-developer-portal-openapi.yml](openapi/choreo-developer-portal-openapi.yml)

### Choreo Insights API

Programmatic access to observability, monitoring, and analytics data for APIs and components deployed on the Choreo platform.

- **Human URL:** [https://wso2.com/choreo/docs/monitoring-and-insights/work-with-choreo-insights-api/access-the-choreo-insights-api/](https://wso2.com/choreo/docs/monitoring-and-insights/work-with-choreo-insights-api/access-the-choreo-insights-api/)
- **OpenAPI:** [openapi/choreo-insights-openapi.yml](openapi/choreo-insights-openapi.yml)
